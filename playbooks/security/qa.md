# Security QA Playbook

## Verification Steps
1. **Scan Coverage**: Were all specified tools run?
2. **Finding Accuracy**: Are reported vulnerabilities confirmed?
3. **Severity Rating**: CVSS scores accurate?
4. **Remediation**: Fix recommendations are actionable?
5. **False Positives**: Obvious false positives filtered out?
6. **OWASP Alignment**: Top 10 categories checked?
7. **Compliance**: Relevant standards addressed (SOC2, PCI, etc.)?

## Scoring
- 0.9+: Thorough scan, accurate findings, actionable fixes
- 0.7-0.8: Good coverage with minor gaps
- <0.7: Missing critical scan categories or inaccurate findings
